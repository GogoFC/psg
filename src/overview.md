# Server Overview - Dell Power Edge R240


Operating System: FreeBSD

ZFS pools: 

- `zroot` - OS pool
- `fantom-pool` - backup pool

Hypervisor: bhyve

Virtual machines:

- `nextcloud-psg` - Nextcloud server production VM
- `guacamole` - Remote desktop access
- `xubuntu` - Ubuntu with Desktop Environment
